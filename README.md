# An Eye for the Sky
A 2d bullet hell wave survival game inspired by cuphead's airplane levels (made for the mini jam 169)

## Controls

wasd or arrow keys to move

left click to fire

## Story
The year is 20XX aliens from the planet Xeronk have decided to attack the united country of sosigia. Take to the skies and beat the Xeronkians using their technology. sadly their technology hasn't been perfectly understood so it both helps and hurts you. See how long you can survive the endless horde!
